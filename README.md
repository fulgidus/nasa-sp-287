# NASA SP-287

A study of [NASA SP-287 — *What Made Apollo a Success?*](https://ntrs.nasa.gov/citations/19720005243)

Contains a parsed/annotated version of the document and a submodule with recreated plots.

## Contents

- `What made Apollo a success.pdf` — original document
- `plots/` — Python/Matplotlib recreations of the figures ([repo](https://github.com/fulgidus/nasa-sp-287-plots), forked from [this repo](https://github.com/ragumagu/apollo-plots))
